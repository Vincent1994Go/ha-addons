# Vincent1994Go HA Add-ons

This repository contains Home Assistant Add-ons maintained by Vincent1994Go.

## Available Add-ons

- netease-cloud-music-api
- phddns

## Add this repository to Home Assistant

1. Open Home Assistant.
2. Go to `Settings` -> `Add-ons` -> `Add-on Store`.
3. Click the `⋮` menu in the top right corner.
4. Select `Repositories`.
5. Add this repository URL:

   `https://github.com/Vincent1994Go/ha-addons`

## Add-on source

- netease-cloud-music-api: https://github.com/neteasecloudmusicapienhanced/api-enhanced
- phddns: https://www.oray.com/

# 🌐 phddns - 花生壳客户端

phddns 是一个用于 Oray 的花生壳服务的 Home Assistant 插件。它支持多种架构，插件可以自动启动并在后台运行。

## ✨ 功能

- 🔄 自动更新动态 DNS
- 🌍 支持多种网络架构
- ⚙️ 简单易用的配置

## 📥 安装

1. 在 Home Assistant 中，打开“Add-ons”页面。
2. 🔍 搜索“phddns”并点击安装。
3. 🛠️ 配置您的花生壳账户信息。
4. ▶️ 启动插件。

## 📖 使用
基础镜像是Ubuntu，所以请参照Ubuntu的说明使用
[请按照花生壳说明使用](https://service.oray.com/question/11630.html)。
